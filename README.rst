==============
sliding-puzzle
==============

A simple sliding puzzle in Tkinter

Requirements:

 * python-imaging-tk

Usage:

.. code:: bash

    ./puzzle.py
    ./puzzle.py --image=/path/to/image --board-grid=6

Keys:

 * Move Up:  ↑ or k
 * Move Down: ↓ or j
 * Move Left: ← or h
 * Move Right: → or l
 * Help: H (Shift + h)
